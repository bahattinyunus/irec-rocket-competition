# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Eğer roketin güvenliğini tehlikeye atabilecek bir açık (örn. şifresiz telemetri linki) bulursanız:

1.  Lütfen **Issue** açmayın (açıkça yayınlamayın).
2.  Bize doğrudan email atın: **security@gokcenrocket.org**
3.  Konu başlığına prefix olarak `[SECURITY]` ekleyin.

Ekibimiz 48 saat içinde size dönecektir. Teşekkürler!
